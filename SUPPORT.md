# Varpa Match Journal Support

For support, use the Profile screen to reset local data or replay onboarding.

Suggested public support contact for App Store Connect:

support@rbtvarparain.app

Varpa Match Journal is a local-first SwiftUI sports companion. It does not require login, subscriptions, or purchases. User-created match and kit content remains on the device; Firebase is used for push notification plumbing and basic analytics diagnostics.
